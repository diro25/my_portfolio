# My Personal Portfolio Website

A clean, responsive, and modern portfolio website built from scratch to showcase my technical skills, academic projects, and engineering journey.

👉 **[Live Demo Viewable Here](YOUR_LIVE_GITHUB_PAGES_LINK_HERE)**

---

## 🛠️ Built With

- **HTML5:** Structural layout and semantic page content.
- **CSS3:** Custom styling, grid layouts, and media queries for mobile responsiveness.
- **JavaScript (Vanilla):** Interactive components, smooth scrolling, and dynamic elements.

## 🚀 Key Features

- **Responsive Design:** Optimized for seamless viewing on desktops, tablets, and mobile phones.
- **Project Showcase:** An interactive grid displaying featured engineering and programming projects.
- **About Me Section:** A brief overview of my educational background, core competencies, and career goals.
- **Contact Form:** A functional layout designed for professional inquiries and networking.
